# 🎵 Lyric Logic

**Lyric Logic** is a full-stack music trivia web app where users can explore 10 different artists and take lyric-based quizzes for each one. It includes a custom backend built with Express.js and a frontend quiz experience that dynamically calculates scores.

---

## 🚨 Built for a Jackathon

This project was created during a **4–5 hour Hackathon-style sprint** with a partner. The goal was to rapidly prototype a creative, music-inspired web experience — from custom API setup to full UI/UX — all within a tight time constraint.

---

## 🎯 Features

- 🎤 View a lineup of 10 musical artists
- 🧠 Take a quiz for each artist based on their lyrics
- 🖼️ Each artist has a photo and metadata pulled from JSON
- 📊 Quiz scoring calculated on the client side
- 🔌 Backend serves data and images using Express and static routes

## 🚀 Getting Started

### 🧪 Backend Setup

1. Navigate to the `server/` folder:

```bash
cd server
```

2. Install dependencies:

```bash
npm install
```

3. Run the server:

```bash
npm run dev
# or
node index.js
```

The server will start at `http://localhost:8080` by default.

---

### 💻 Frontend Setup

1. Open the `client/index.html` file

2. **(Optional)** Use the **Live Server** extension in VS Code:

   - Right-click `index.html`
   - Click **“Open with Live Server”**
